CSS3: SELECTORES CSS

Esta es la definicion de un CSS Selector:

Los selectores CSS se utilizan para seleccionar el contenido al que desea aplicar estilo. Los selectores son parte del conjunto de reglas CSS. Los selectores CSS seleccionan elementos HTML de acuerdo con su id, clase, tipo, atributo, etc. Hay varios tipos diferentes de selectores en CSS.

Podemos dividir los selectores de CSS en cinco categorías: Simple Selectors, Combinator Selectors, Pseudo-class Selectors, Pseudo-elements Selectors, Attribute Selectors.

Principlamente en este OVI se enseñará la estructura, ejemplos y vídeos relacionados con los selectores simples, basicos o generales del mundo de CSS. Explicando: Element Selector, Id Selector, Class Selector, Universal Selector y Grouping Selector.

CSS Id Selector

El selector de id utiliza el atributo id de un elemento HTML para seleccionar un elemento específico. La identificación de un elemento es única dentro de una página, por lo que el selector de identificación se utiliza para seleccionar un elemento único. Para seleccionar un elemento con una identificación específica, escriba un carácter de almohadilla (#), seguido de la identificación del elemento.

CSS Class Selector

El selector de clases selecciona elementos HTML con un atributo de clase específico. Para seleccionar elementos con una clase específica, escriba un punto (.), Seguido del nombre de la clase.

También se puede especificar que solo los elementos HTML específicos se vean afectados por una clase. Los elementos HTML también pueden hacer referencia a más de una clase.

CSS Grouping Selector

El selector de agrupación selecciona todos los elementos HTML con las mismas definiciones de estilo. Será mejor agrupar los selectores para minimizar el código. Para agrupar selectores, separe cada selector con una coma.
